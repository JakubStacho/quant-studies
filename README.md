# quant-studies

A personal sandbox for learning and exploring quantitative finance concepts through code. Projects here range from walkthroughs to scrappy experiments.

## Explorations

| Folder | Topic |
|--------|-------|
| [`portfolio-optimization/`](portfolio-optimization/) | Mean-variance optimization, efficient frontier, and the practical limits of MVO |

## Setup

```bash
python -m venv venv-quant
source venv-quant/bin/activate
pip install -r requirements.txt
```
